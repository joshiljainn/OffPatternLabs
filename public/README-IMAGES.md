# Image Assets Guide

This folder contains all image assets for the OffPattern Labs website.

## Required Images

### Logo
- **File**: `logo.svg` or `logo.png`
- **Recommended size**: SVG (scalable) or PNG at 2x resolution
- **Usage**: Header/navigation area

### Open Graph Image
- **File**: `og-image.png`
- **Size**: 1200 x 630 pixels
- **Usage**: Social media sharing preview

### Project Images (Optional)
Place project/case study images in the `/images` folder:
- `images/project-1.jpg`
- `images/project-2.jpg`
- etc.

### Workspace/Brand Images
For future use:
- Team photos
- Workspace images
- Client testimonial photos
- Product shots

## Image Optimization Tips

1. **Format**: Use WebP or AVIF for modern browsers (Next.js handles this automatically)
2. **Size**: Keep images under 300KB when possible
3. **Dimensions**: Export at 2x the display size for retina displays
4. **Naming**: Use descriptive, lowercase names with hyphens (e.g., `hero-background.jpg`)
